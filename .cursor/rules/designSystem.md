# デザインシステム: コア要素

このドキュメントでは、プロジェクト全体で一貫して使用するデザインシステムのコア要素を定義します。

## 1. ベースデザインガイドライン

### 1.1 選定したデザインガイドライン

Human Interface Guidelines (HIG) をベースとしたデザインシステムを採用しています。これにより、モダンで洗練された印象を持つシンプルなデザインを実現します。

### 1.2 選定理由

- シンプルさと洗練性が特徴
- モダンな印象を与えるのに最適
- 直感的なユーザーインターフェイス
- 一貫性のあるデザイン言語

### 1.3 アイコンシステム

アイコンシステムについては今後検討予定です。

## 3. レスポンシブデザイン

### 3.1 ブレイクポイント

Tailwind のデフォルトブレイクポイントを採用:

- sm: 640px
- md: 768px
- lg: 1024px
- xl: 1280px
- 2xl: 1536px

### 3.2 レスポンシブ設計の原則

- モバイルファーストアプローチ
- 流動的レイアウト
- コンテンツの優先順位付け

## 4. アクセシビリティ

### 4.1 色のコントラスト

- WCAG AA レベルの準拠を目指す
- コントラスト比 4.5:1 以上を確保

### 4.2 キーボードアクセシビリティ

- フォーカス状態の視覚化（focus-visible:ring-2）
- タブ順序の論理的な設計

### 4.3 スクリーンリーダー対応

- aria 属性の適切な使用
- セマンティック HTML の活用
- 視覚的に隠れた要素のための visually-hidden ユーティリティクラス

## 5. コアコンポーネント

### 5.1 ボタン (Button)

- **バリエーション**:

  - primary: プライマリアクション用
  - secondary: セカンダリアクション用
  - outline: アウトラインスタイル
  - ghost: 背景なしスタイル
  - link: リンクスタイル

- **サイズ**:
  - sm: 小サイズ
  - md: 中サイズ (デフォルト)
  - lg: 大サイズ
  - icon: アイコン用

### 5.2 入力フィールド (Input)

- エラー状態表示
- ヘルパーテキスト
- 無効状態のスタイリング

### 5.3 カード (Card)

- **バリエーション**:

  - default: 標準カード（影付き）
  - outline: アウトラインカード（ボーダー付き）
  - flat: フラットカード（背景色のみ）

- **サブコンポーネント**:
  - CardHeader: カードヘッダー（白背景）
  - CardTitle: カードタイトル（固定高さh-14、2行制限line-clamp-2）
  - CardDescription: カード説明
  - CardContent: カードコンテンツ（ライトグレー背景）
  - CardFooter: カードフッター（白背景）

- **特殊カード**:
  - EventCard: イベント情報表示用カード
    - 白背景ヘッダー（カテゴリーバッジを含む）
    - ライトグレー背景のコンテンツ
    - 白背景フッター（価格表示を含む）
    - 一貫した高さのタイトル表示（固定高さh-14）

### 5.4 バッジ (Badge)

- **バリエーション**:

  - default: プライマリカラーベース
  - secondary: セカンダリカラーベース
  - success: 成功表示用
  - warning: 警告表示用
  - error: エラー表示用
  - info: 情報表示用

- **スタイル**:
  - 標準: 塗りつぶし
  - outline: アウトライン

### 5.5 検索UI (SearchFilter)

- **バリエーション**:
  - デスクトップ: カテゴリードロップダウン + 検索フィールド + 検索ボタン
  - モバイル: シンプル化された検索フィールド + 検索ボタン（カテゴリードロップダウンは非表示）

- **レスポンシブ動作**:
  - md:ブレークポイント未満ではカテゴリードロップダウンを非表示
  - プレースホルダーはモバイルでもデスクトップでも「カテゴリーやイベントを検索」に統一
